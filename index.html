📝 Original Java Code

public class ScopeRules
{
    // CLASS LEVEL SCOPE
    static final double rate = 10.50;
    static int z;
    static double t;
    static int w;
    
    // METHOD main()
    public static void main(String[] args)
    {
        int num;
        double x;
        double z;
        char ch;
    }
    
    // METHOD one()
    public static void one(int x, char y)
    {
        // Method body
    }
    
    // METHOD two()
    public static void two(int one, int z)
    {
        char ch;
        int a;
        
        // BLOCK THREE
        {
            int x = 12;
        }
    }
}

📌 Summary by Scope Type

🏛️ Class Scope

Visible everywhere in the class

  • rate - static final double
  • z - static int (can be hidden)
  • t - static double
  • w - static int
  • main() - method
  • one() - method
  • two() - method

📱 Method Scope

Visible only within specific methods

main():
  • args (parameter)
  • num (local)
  • x (local)
  • z (local - shadows class z)
  • ch (local)
one():
  • x (parameter)
  • y (parameter)
two():
  • one (parameter - shadows method)
  • z (parameter - shadows class z)
  • ch (local)
  • a (local)

📦 Block Scope

Visible only within specific blocks

Block three (inside two()):
  • x = 12 (block local)

✅ This block can also see all of two()'s scope

❌ Variables from block not visible outside

⚠️ Shadowing/Hiding Examples

1. Class variable 'z' is shadowed:
  • By local variable 'z' in main() method
  • By parameter 'z' in two() method
2. Method name 'one' is shadowed:
  • By parameter 'one' in two() method
✅ Accessibility Rules:
  • Class variables: Accessible everywhere (unless shadowed)
  • Method parameters: Accessible only in their method
  • Local variables: Accessible only from declaration to end of method/block
  • Block variables: Most restricted scope
